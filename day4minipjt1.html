Installation

Download the installer from the official website and run the setup file. Follow the on-screen instructions to complete the installation process.

Usage

After installation, open SucheeApp from your applications folder. Configure your preferences and begin using the main dashboard features.

Troubleshooting

Problem: Application Will Not Start

Solution: Ensure your system meets the minimum requirements. Try restarting your device and reinstalling the application if necessary.

Problem: Unable to Connect to Server

Solution: Check your internet connection and firewall settings. Verify that the server address is entered correctly in the settings menu.
